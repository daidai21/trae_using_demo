# 电商交易平台开发计划

## 一、项目概述

### 1.1 项目目标

从0到1开发一个功能完整的电商交易平台，包含商家管理、商品管理、营销、购物车、下单、订单管理、物流、结算等核心功能。

### 1.2 技术栈

* **后端**: Golang

* **前端**: React + Ant Design

* **数据库**: SQLite + Redis

* **API通信**: RESTful API

***

## 二、项目架构设计

### 2.1 目录结构

```

│   ├── cmd/             # 应用入口
│   │   └── server/

│   │   ├── handler/     # HTTP处理器
│   │   ├── service/     # 业务逻辑层
```

### 2.2 数据库设计

核心数据表：

* `users` - 用户表

* `merchants` - 商家表

* `products` - 商品表

* `carts` - 购物车表

* `orders` - 订单表

* `order_items` - 订单详情表

## 三、功能模块设计

### 3.1 用户认证模块

* 用户注册/登录

* JWT Token 认证

* 密码加密存储

### 3.2 商家管理模块

* 商家入驻

* 商家信息编辑

* 商家列表查询

### 3.3 商品管理模块

* 商品发布

* 商品列表（含搜索、筛选）

* 商品详情

* 商品库存管理

### 3.4 购物车模块

* 添加商品到购物车

* 购物车列表

* 修改商品数量

* 删除购物车商品

### 3.5 订单管理模块

* 创建订单

* 订单列表（我的订单）

* 订单详情

* 订单状态更新

## 四、开发步骤

### 阶段一：项目初始化

1. 创建项目目录结构
2. 初始化 Go 项目（go mod）
3. 初始化 React 项目（Create React App 或 Vite）
4. 配置基础依赖

### 阶段二：后端开发

1. 设计并创建数据库表结构
2. 实现用户认证 API（注册、登录）
3. 实现商家管理 API
4. 实现商品管理 API
5. 实现购物车 API
6. 实现订单管理 API
7. 添加 CORS 中间件

### 阶段三：前端开发

1. 配置 React Router 路由
2. 实现用户登录/注册页面
3. 实现商家管理页面
4. 实现商品列表和详情页面
5. 实现购物车页面
6. 实现订单管理页面
7. 集成 Ant Design 组件库

### 阶段四：联调与测试

1. 前后端接口联调
2. 功能测试
3. 修复 Bug

***

## 五、关键技术点

### 后端（Golang）

* Web 框架：Hertz + Kitex

* ORM：GORM

* JWT：github.com/golang-jwt/jwt

* 密码加密：bcrypt

### 前端（React + Ant Design）

* 构建工具：Vite（推荐）或 Create React App

* 状态管理：React Context API 或 Zustand

* HTTP 客户端：Axios

* 路由：React Router v6

***
