# 电商交易平台 - 产品需求文档

## 概述
- **Summary**: 从0到1开发一个功能完整的电商交易平台，包含用户认证、商家管理、商品管理、购物车、下单、订单管理等核心功能，采用前后端分离架构。
- **Purpose**: 构建一个可运行的电商平台演示项目，展示全栈开发能力，提供商家和用户两个角色的完整购物流程体验。
- **Target Users**: 普通购物用户、商家用户

## 目标
- 实现完整的用户注册/登录流程，使用JWT认证
- 实现商家管理功能，包括商家入驻和信息编辑
- 实现商品管理功能，支持商品发布、列表展示、详情查看
- 实现购物车功能，支持添加、修改数量、删除商品
- 实现订单管理功能，支持创建订单、查看订单列表和详情、订单状态更新
- 提供美观易用的React前端界面，使用Ant Design组件库

## 非目标（范围外）
- 不实现支付功能
- 不实现物流追踪功能
- 不实现复杂的营销活动（优惠券、秒杀等）
- 不实现用户评价功能
- 不实现Redis缓存
- 不实现复杂的权限管理系统

## 背景与上下文
- 这是一个学习演示项目，用于展示从0到1构建电商平台的完整流程
- 技术栈已确定：后端Golang + 前端React + Ant Design + SQLite数据库
- 采用前后端分离架构，通过RESTful API通信

## 功能需求
- **FR-1**: 用户可以注册新账号并登录系统
- **FR-2**: 用户可以浏览商家列表和商品列表
- **FR-3**: 用户可以查看商品详情
- **FR-4**: 用户可以将商品添加到购物车
- **FR-5**: 用户可以管理购物车（修改数量、删除商品）
- **FR-6**: 用户可以从购物车创建订单
- **FR-7**: 用户可以查看自己的订单列表和订单详情
- **FR-8**: 商家可以入驻平台并编辑商家信息
- **FR-9**: 商家可以发布和管理商品

## 非功能需求
- **NFR-1**: 系统响应时间：页面加载时间 < 2秒，API响应时间 < 500ms
- **NFR-2**: 安全性：密码使用bcrypt加密存储，JWT Token有效期24小时
- **NFR-3**: 可用性：前端界面使用Ant Design，提供良好的用户体验
- **NFR-4**: 可维护性：代码结构清晰，遵循最佳实践

## 约束
- **技术**: 后端使用Golang、前端使用React + Ant Design、数据库使用SQLite
- **业务**: 这是一个演示项目，功能范围有限
- **依赖**: Golang 1.21+, Node.js 18+, React 18+, Ant Design 5.x

## 假设
- 运行环境已安装Golang和Node.js
- 用户具有基本的计算机操作能力
- 本地开发环境网络连接正常

## 验收标准

### AC-1: 用户注册与登录
- **Given**: 用户访问平台
- **When**: 用户填写注册信息并提交，或使用已注册账号登录
- **Then**: 系统创建新账号或验证登录成功，返回JWT Token
- **Verification**: `programmatic`
- **Notes**: 密码需加密存储，Token需包含用户ID和过期时间

### AC-2: 商家入驻
- **Given**: 已登录用户
- **When**: 用户提交商家入驻申请
- **Then**: 系统创建商家记录，关联到当前用户
- **Verification**: `programmatic`

### AC-3: 商品发布
- **Given**: 已入驻的商家用户
- **When**: 商家填写商品信息并发布
- **Then**: 系统创建商品记录，关联到该商家
- **Verification**: `programmatic`

### AC-4: 商品浏览
- **Given**: 任何用户（登录或未登录）
- **When**: 用户访问商品列表或商品详情页
- **Then**: 系统展示商品列表或商品详情信息
- **Verification**: `programmatic`

### AC-5: 购物车管理
- **Given**: 已登录用户
- **When**: 用户添加商品到购物车、修改数量或删除商品
- **Then**: 系统更新购物车数据并展示最新状态
- **Verification**: `programmatic`

### AC-6: 订单创建
- **Given**: 已登录用户且购物车有商品
- **When**: 用户提交订单
- **Then**: 系统创建订单记录，扣减库存，清空购物车
- **Verification**: `programmatic`

### AC-7: 订单查询
- **Given**: 已登录用户
- **When**: 用户访问订单列表或订单详情页
- **Then**: 系统展示该用户的订单信息
- **Verification**: `programmatic`

### AC-8: 前端界面
- **Given**: 用户访问前端应用
- **When**: 用户在各页面间导航和操作
- **Then**: 界面响应流畅，Ant Design组件正确渲染，用户体验良好
- **Verification**: `human-judgment`
